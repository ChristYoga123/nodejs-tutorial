# Apa itu Node js

Sumber: nodejs.org

### Pengertian

NodeJS => Javascript berbasis server-side yang dijalankan melalui runtime build yaitu V8. 

Nodejs memberikan tambahan kelebihan pada Javascript sehingga javascript yang awalnya hanya digunakan untuk membuat website interaktif saja menjadi mampu untuk mengolah file pada PC, koneksi ke Database dan sebagainya.

### Kenapa NodeJS

1. Nodejs menggunakan NonBlocking I/O artinya tidak perlu menunggu sebuah request selesai terlebih dahulu untuk melayani request berikutnya. NodeJs menjalankan tugas secara paralel. Hal ini membuat nodejs dapat menerima banyak request dengan cepat.
2. Library dan komunitas yang lengkap dan besar.
3. Bisa membangun API berbasis REST yang dapat diakses banyak aplikasi.
4. Sering digunakan membuat aplikasi real-time seperti chatting, IOT, dll.

### Instalasi Nodejs

Download Nodejs versi LTS yang lebih stabil di nodejs.org
